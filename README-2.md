# LILA BLACK — Player Journey Visualizer

A browser-based visualization tool for exploring player behavior across LILA BLACK's three maps. Built as part of the APM assignment.

**Live tool:** https://dynamic-figolla-efafa9.netlify.app

---

## What it does

- Displays player paths on the correct minimap for each match
- Distinguishes human players (blue) from bots (grey)
- Shows kill, death, loot, and storm death events as markers
- Filter by map, day, and individual match
- Timeline playback to watch a match unfold
- Heatmap overlays for traffic, kill zones, death zones, and loot zones

---

## How to run locally

**Step 1 — Process the data**

You'll need the `player_data` files from Lila Games (not included in this repo as they contain proprietary game data).

Install dependencies:
```
pip3 install pyarrow pandas
```

Place the `February_10` through `February_14` folders and `minimaps/` in the root directory, then run:
```
python3 process_data.py
```

This generates the `data/` folder with processed JSON files.

**Step 2 — Start a local server**
```
python3 -m http.server 8080
```

Then open **http://localhost:8080/index.html**
